# NC//DATABANK

Фанатская энциклопедия по вселенной Cyberpunk 2077 в стиле википедии. Статичный сайт без сборки: чистые HTML/CSS/JS.

## Запуск

Откройте `index.html` в браузере — или запустите локальный сервер:

```bash
npm start          # npx serve .
# или
python3 -m http.server 8080
```

## Структура

```
index.html      — каркас страницы (шапка, сайдбар, контент)
css/styles.css  — вся стилистика (палитра и токены в :root)
js/data.js      — контент: категории и статьи
js/app.js       — hash-роутер, поиск, рендер страниц
```

## Как добавить статью

Добавьте объект в массив `DB.articles` в `js/data.js`:

```js
{
  id: "moya-statya",            // уникальный id — попадает в URL: #/article/moya-statya
  title: "Моя статья",
  category: "tech",             // id из DB.categories
  subtitle: "Одно предложение для карточки и шапки статьи.",
  featured: true,               // необязательно: показать на главной
  infobox: { "Ключ": "Значение" },
  sections: [
    { heading: "Обзор", html: `<p>Текст. Ссылка на другую статью:
      <a href="#/article/v">Ви</a></p>` }
  ],
  related: ["v", "night-city"]  // id связанных статей
}
```

Сайдбар, поиск, категории и счётчики обновятся автоматически.
